This project is part of the **IE Sustainability Datathon (March 2026)**, developing an optimal placement strategy for EV charging infrastructure along Spain's interurban highway network for the 2027 horizon.

The solution combines geospatial analysis of road networks, traffic patterns, and existing charger coverage with mathematical optimization (set cover formulation) to propose new station locations — subject to real grid capacity constraints from three distribution system operators (i-DE, Endesa, Viesgo).

**Expected outputs:** interactive Folium map of proposed stations, grid friction analysis, and a network-wide KPI dashboard.
